Projeto desenvolvido para disciplina de aplicativos web e mobile

## requisitos

- node >= **16.17.0**
- yarn >= **1.22.19**
- sdk **android** ou **ios**

## como executar

Entre nas pastas _web_, _mobile_ e _server_ e execute: `yarn install && yarn start`
